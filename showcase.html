Plane Management System

Project Description: Plane Management System

The Plane Management System is a Java-based application designed to facilitate the management of seat bookings for a fictional airline. It enables users to buy and cancel seats, find the first available seat, display the seating plan, print ticket information, and search for existing tickets. The application aims to streamline the booking process, making it user-friendly for passengers and efficient for airline management.

Key Features:
Seat Booking: Allows users to reserve seats based on row (A-D) and seat number (1-14).
Seat Cancellation: Users can cancel previously booked seats, updating the seating availability in real-time.
Available Seats Finder: Quickly identifies and displays the first available seat in the cabin.
Seating Plan Display: Visually represents the seating layout, indicating sold and available seats.
Ticket Information Management: Prints ticket details and saves them to text files for record-keeping.
Ticket Search: Enables users to check the status of a seat and retrieve associated ticket information.

Technologies Used:
Java: The core programming language used to develop the application, providing object-oriented programming features.
Scanner Class: Utilized for user input, allowing interaction through the console.
File I/O: Implemented to handle ticket information storage, enabling ticket data to be saved to text files.
This project demonstrates practical applications of object-oriented programming, user input validation, and file management in Java, providing a comprehensive solution for seat management in an airline context.

Inky, Pinky Polly game


Inky, Pinky, Polly is an interactive game designed for two players, where each player simultaneously selects one of three character names: Inky, Pinky, or Polly. The game is a fun twist on classic hand games like Rock, Paper, Scissors, providing an engaging way to determine a winner through simple mechanics.

Game Mechanics:
Character Strengths:
Inky beats Polly (Inky is stronger than Polly).
Pinky beats Inky (Pinky is stronger than Inky).
Polly beats Pinky (Polly is stronger than Pinky).
If both players choose the same character, the game results in a tie and is immediately replayed until a winner is determined.
Key Features:
User-Friendly Interface: Players can easily select their character names with a simple click or tap.
Real-Time Results: The game provides immediate feedback on the outcome of each round, displaying the winner or indicating a tie.
Replay Option: After each round, players can continue playing or exit the game.
Technologies Used: Python

Academic Progression Predictor

Project Description: Academic Progression Prediction Program

The Academic Progression Prediction Program is a Python-based application designed to help university staff predict the progression outcomes of students at the end of each academic year based on their credit scores in three categories: pass, defer, and fail. This program utilizes user-defined functions for modularity and clarity, making the code easy to read and maintain.

Key Features:
User Input: The program prompts the user to input the number of credits for pass, defer, and fail categories. It includes validation to ensure the inputs are integers within specified ranges and sum correctly to 120.
Progression Outcomes: Based on the input, the program determines the progression outcome using a predefined set of rules from the university's regulations, such as "Progress," "Module Trailer," "Module Retriever," and "Exclude."
Multiple Predictions: Users can input data for multiple students in a loop until they choose to quit, allowing for efficient handling of numerous cases.
Histogram Visualization: Upon exiting the loop, the program generates a histogram displaying the count of students in each progression category using the graphics.py module.
Data Storage: The program saves the input progression data in a list, allowing for easy access and display of the results.
Text File Output: Optionally, the program can save the progression data to a text file for future reference.
Technologies Used:
Programming Language: Python
